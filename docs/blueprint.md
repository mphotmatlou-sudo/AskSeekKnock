# **App Name**: A.S.K.

## Core Features:

- Prayer Template: Provide customizable prayer session templates that provide a starting point for structuring prayer.
- Adjustable Timer: Implement an adjustable timer with alarm to guide users through prayer points.
- Prayer Journal: Enable users to create date-stamped journal entries for reflections and insights.
- Bible Verse Integration: Integrate a direct linking mechanism (a 'tool') that allows the LLM to pull Bible verses from BibleHub or BibleGateway.
- Template Suggestions: Offer a platform for users to suggest and crowdsource new prayer directions and template ideas.
- Dashboard: Display a dashboard summarizing journal entries, commonly used templates, and other stats, for the purpose of review and reflection on their spiritual journey.

## Style Guidelines:

- Primary color: A serene light-purple (#B19CD9) to evoke a sense of spirituality and peace.
- Background color: Very light grayish-purple (#F4F0F9), offering a calm backdrop that does not distract.
- Accent color: A gentle lavender (#D0B4DE) for interactive elements and highlights.
- Body and headline font: 'Alegreya', a serif font for readability and a contemporary feel.
- Use simple, line-based icons to represent different prayer categories and actions.
- Employ a clean and minimalist design to minimize distractions and facilitate focus on prayer.
- Use subtle transitions for navigation and interactive elements.
